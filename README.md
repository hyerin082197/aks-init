# Azure Kubernetes Service

Init

## Prerequisites

- Install terraform: <https://developer.hashicorp.com/terraform/tutorials/aws-get-started/install-cli#install-terraform>

## Quick Start

`main.tf` 위치하는 directory 에서 실행

```sh
### terraform provider install
terraform init

### plan 검증 및 생성
terraform plan

### 생성된 plan 기준으로 적용
terraform apply
```
